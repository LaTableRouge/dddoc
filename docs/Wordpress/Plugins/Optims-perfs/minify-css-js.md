**WP Super Minify** is a WordPress plugin that combines, minifies, and caches JavaScript and CSS files to optimize your site's performance.

## Installation

### Installation via WordPress Dashboard

1. Go to your WordPress dashboard.
2. Navigate to **Plugins > Add New**.
3. Search for "**WP Super Minify**".
4. Click **Install Now** and then **Activate**.

### Installation via Composer

If you're using **Composer** and **Wpackagist**, you can install the plugin with the following command:

```bash
composer require wpackagist-plugin/wp-super-minify
