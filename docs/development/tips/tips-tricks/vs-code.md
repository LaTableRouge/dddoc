When adding 410 or 301 rules, be careful not to duplicate the same rule.

VS Code can make this task much easier with two very useful commands to run on a redirect conf file:
* Shift + Cmd + P => Sort Lines Ascending
* Shift + Cmd + P => Delete Duplicate Lines
